# 원주시 버스 종점 출발 시간 정보 사이트 개발 계획

## 프로젝트 개요

원주시 버스의 종점 출발 시간을 확인하고, 현재 시간과 비교하여 다음 출발까지 남은 시간을 확인할 수 있는 웹 사이트입니다. 모바일 최적화된 반응형 디자인으로 구현됩니다. 모든 데이터는 public 폴더의 정적 파일에서 로드합니다.

## 기술 스택

- Next.js
- TailwindCSS
- pnpm (패키지 관리자)
- FSD (Feature-Sliced Design) 아키텍처

## 기능 명세

### 1. 메인 페이지

- 사이트 소개 섹션 (원주시 버스 종점 출발 시간 정보)
- 종점 목록 표시 (카드 형태)
- 종점 검색 기능
- 현재 시간 표시

### 2. 종점 상세 페이지

- 종점 이름 및 기본 정보 표시
- 오늘의 요일 및 특수일(공휴일, 방학) 상태 표시
- 시간순으로 정렬된 버스 노선 목록
- 각 노선별 정보 표시:
  - 노선 번호
  - 배차 간격
  - 첫차/막차 시간
  - 남은 버스 수
  - 다음 출발까지 남은 시간

### 3. 노선 상세 페이지

- 노선 번호, 출발지, 도착지 정보
- 전체 운행 일정 표시
- 해당 노선의 전체 정류장 목록 (가능한 경우)
- 첫차/막차 시간 및 배차 간격
- 요일별 운행 정보 차이점 표시

### 4. 시간 계산 기능

- 현재 시간과 다음 출발 시간 비교
- 남은 시간을 분 단위로 계산
- 이미 출발한 버스와 아직 출발하지 않은 버스 구분
- 오늘 남은 버스 수 계산

### 5. 요일 및 특수일 판별 기능

- 현재 날짜를 기준으로 요일 판별
- 2025년 기준 공휴일 데이터 내장
- 방학 기간 설정:
  - 여름방학: 6월 20일 ~ 8월 30일
  - 겨울방학: 12월 31일 ~ 2월 28일

## FSD 아키텍처 구조

```
src/
├── app/              # 페이지 및 라우팅
├── widgets/          # 복합 UI 컴포넌트
├── features/         # 비즈니스 로직 단위
├── entities/         # 비즈니스 엔티티
└── shared/           # 공통 유틸리티 및 UI
```
